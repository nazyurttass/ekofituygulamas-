# EcoFit

## Proje Amacı
Bu proje, Java programlama dili ve MySQL veritabanı kullanılarak geliştirilmiş konsol tabanlı bir fitness uygulamasıdır. 

Uygulamanın amacı, spor salonu üyelerinin:
- Güvenli şekilde sisteme giriş yapabilmesi,
- Kalori hesaplamaları yapabilmesi,
- Kendilerine uygun egzersiz programları önerileri alabilmesi,
- Kullanıcı bilgilerinin güvenli biçimde saklanabilmesi

ve kişiselleştirilmiş bir spor deneyimi yaşamalarını sağlamaktır.

## Kullanılan Teknolojiler
- Java
- MySQL

## Özellikler
- Kullanıcı kayıt ve giriş sistemi
- Kalori hesaplama
- Egzersiz öneri sistemi
- Veritabanı bağlantısı
- Kişiye özel kullanıcı deneyimi
